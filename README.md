E-Commerce Business Performance Dashboard

An interactive Power BI dashboard analyzing e-commerce sales, profitability, and customer behavior to support faster, data-backed business decisions.

Overview

This dashboard analyzes 30,000+ orders across 8,683 customers, tracking:


₹11.37M in total order value
₹2.22M in revenue
Revenue trends by month
Profit by region
Order fulfillment status
Customer segmentation (New, Returning, Loyal, Premium)


Drill-through filters allow deeper exploration by season, order status, gender, and shipping method.

Key Insight

Electronics emerged as the top revenue-generating category, contributing ₹63.2 lakh in revenue at a 76% profit margin — the dashboard surfaced clear patterns in customer behavior, order fulfillment, and seasonal demand that support faster, data-backed decision-making.
